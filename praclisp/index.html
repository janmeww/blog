Practice for Clojure

Suppose you know some programming in any language, but now you want learn Clojure–which I highly recommend. There are a number of good books written on Clojure, but none of them allows a learner to adjust to Lispy goodness.

By the way, Clojure is dialect of the Lisp programming language, the most important programming language ever. (Feel free to Google why.)

And Clojure is the most important Lisp, after Common Lisp. The only catch is, if you have never coded in Lisp or Scheme, you might find the transition to the Lisp way a bit strange.

Unfortunately, many texts on Clojure do not provide enough transition material, but Clojure, like all Lisp dialects, requires that you get used to its syntax and how it works.

To this end, it does not matter which Lisp dialect you practice in. The recommended one is Scheme though, since it is the most suitable for an introductory exposure to Lisp and also to programming.

Therefore, here is my free, online Scheme practice drills which start from 0 and go all the way to tail recursion. My goal was to create practice problems that are slightly more difficult, varied, and in-depth than the ones you’d encounter in any introductory Scheme or Lisp programming text.

Thus, if you are completely new to programming, I suggest you start with Brian Harvey’s Simply Scheme, and finish the first three chapters. After that, you will be able to work on the problems presented in my drillbook.

If you have done some programming before, you will have no problem diving right in.
